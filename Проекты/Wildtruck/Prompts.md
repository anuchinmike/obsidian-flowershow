## Подкаталоги
```
приступаем к верстке каталога {subcatalog_name} сначала сверстай секцию категории маленькие карточки, названия подразделов каталога возьми из скриншота, адреса файлов картинок из второго скриншота, адреса каталогов сделай транслитерацией их названий. Ничего не меняй в самой верстке, чтобы сохранить консистентность дизайна сайта. Меняй только текстовое наполнение, ссылки и метатеги. Создавай сразу SEO оптимизированные тексты с оптимизацией под ключевые фразы, содержащиеся в названиях каталогов. Образец верстки с маленькими карточками есть в заметках
```

```
теперь сверстай поочередно каждый из подкаталогов. Ничего не меняй в самой верстке, чтобы сохранить консистентность дизайна сайта. Меняй только текстовое наполнение, ссылки и метатеги. Создавай сразу SEO оптимизированные тексты с оптимизацией под ключевые фразы, содержащиеся в названиях каталогов. Образец верстки в заметке "Образец подраздела каталога"
```

## Коренной каталог
```
Теперь сверстай раздел коренного каталога {catalog_name } Ничего не меняй в самой верстке, чтобы сохранить консистентность дизайна сайта. Меняй только текстовое наполнение адреса и метатеги. Создавай сразу SEO оптимизированные тексты с оптимизацией под ключевые фразы, содержащиеся в названиях каталогов. Образец возьми в заметке "Образец верстки для коренного раздела каталога"
```
